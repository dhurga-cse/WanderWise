## рҹҢҲ Visual Hierarchy

### Level 1: Hero/Headers
- Size: 3-5rem (48-80px)
- Weight: Bold (700-800)
- Color: Gradient or Dark Gray

### Level 2: Section Titles
- Size: 2-3rem (32-48px)
- Weight: Bold (700)
- Color: Dark Gray

### Level 3: Card Titles
- Size: 1.5rem (24px)
- Weight: Semibold (600)
- Color: Dark Gray

### Level 4: Body Text
- Size: 1rem (16px)
- Weight: Regular (400)
- Color: Medium Gray

### Level 5: Small Text
- Size: 0.875rem (14px)
- Weight: Regular (400)
- Color: Light Gray

## рҹҺӘ Interactive Elements

### Buttons
- **Primary:** Gradient background, white text, shadow on hover
- **Secondary:** Transparent with border, hover fill
- **Danger:** Red background, white text
- **Icon:** Icon only, circular, hover background

### Inputs
- **Default:** Gray border, rounded corners
- **Focus:** Blue ring, border disappears
- **Error:** Red border, red text below
- **Success:** Green border, checkmark icon

### Cards
- **Default:** White background, subtle shadow
- **Hover:** Lift up, larger shadow
- **Active:** Pressed down effect
- **Selected:** Blue border
